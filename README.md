# Завдання 7: Метод Монте-Карло для симуляції кидання кубиків

## Опис завдання
Програма імітує велику кількість кидків двох кубиків і обчислює ймовірності всіх можливих сум (від 2 до 12) методом Монте-Карло.

## Результати симуляцій

Проведено симуляції з різною кількістю кидків: 100, 1,000, 10,000 та 100,000.

### Порівняння точності

| Кількість кидків | Середня різниця (%) |
|------------------|---------------------|
| 100              | ~3-5%               |
| 1,000            | ~1-2%               |
| 10,000           | ~0.3-0.7%           |
| 100,000          | ~0.1-0.3%           |

## Висновки

1. **Точність методу Монте-Карло:**
   - При збільшенні кількості симуляцій результати наближаються до теоретичних значень
   - При 100,000 кидках різниця з аналітичними розрахунками становить менше 0.3%

2. **Підтвердження закону великих чисел:**
   - Метод Монте-Карло демонструє збіжність до точних значень
   - Чим більше симуляцій, тим вища точність

3. **Найбільш ймовірна сума:**
   - Сума **7** має найвищу ймовірність (~16.67%)
   - Це відповідає аналітичним розрахункам (6 комбінацій з 36 можливих)

4. **Симетрія розподілу:**
   - Ймовірності симетричні відносно суми 7
   - Суми 2 і 12 мають найменшу ймовірність (~2.78%)

5. **Правильність розрахунків:**
   - Порівняння з аналітичними даними підтверджує коректність симуляції
   - Метод Монте-Карло є надійним інструментом для розрахунку ймовірностей
